规则说明

1. 点击一个蓝莓棋子，再点击第二个蓝莓棋子即可交换；也支持手指直接滑动交换。

2. 只有交换后形成 3 个及以上同类棋子连线，才会自动消除并补位；否则交换无效并恢复。

3. 游戏总时长 3 分钟，共 5 关，目标依次为 12、20、28、36、45。

4. 完成第 5 关后，按通关用时进入排行榜，用时越短排名越高。

5. 用户提交成绩时必须填写手机号；同一手机号当天可多次挑战，但排行榜只保留当天最好成绩。

6. 每天 0 点可对上一自然日成绩进行结算，生成前十名单。
